# Font Installation Instructions

This project uses the Poppins font family. Please download and place the following font files in this directory:

1. **Poppins-Regular.ttf**
2. **Poppins-Medium.ttf**
3. **Poppins-SemiBold.ttf**
4. **Poppins-Bold.ttf**

## Download Source

Download Poppins from Google Fonts:
https://fonts.google.com/specimen/Poppins

## Installation Steps

1. Visit the Google Fonts link above
2. Click "Download family" button
3. Extract the ZIP file
4. Copy the required .ttf files to this directory:
   - `frontend/assets/fonts/Poppins-Regular.ttf`
   - `frontend/assets/fonts/Poppins-Medium.ttf`
   - `frontend/assets/fonts/Poppins-SemiBold.ttf`
   - `frontend/assets/fonts/Poppins-Bold.ttf`

After adding the fonts, the app will load them automatically on startup.
